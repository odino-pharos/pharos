## 🇮🇹 Italiano

### Cos'è Φάρος

Φάρος è un'app Android gratuita e open source che **blocca automaticamente il traffico dati mobile** mentre guidi, attraverso una VPN puramente locale. Nessun server, nessuna raccolta dati, nessuna pubblicità.

Come un faro: non insegue, non suona, non pretende. Sta lì, fa la sua luce, e chi naviga la usa quando serve.

### Come funziona

- Rileva la guida tramite GPS: velocità sopra soglia per 15 secondi **e** spostamento reale di almeno 100 metri (doppia condizione contro le false attivazioni da fermo).
- Attiva una VPN locale che blocca il traffico dati di social, messaggi e video.
- **Navigatori e app musicali restano sempre liberi** — criterio "orecchie sì, occhi no": è ammesso ciò che si usa senza distogliere lo sguardo dalla strada.
- Un annuncio vocale calmo ti avvisa quando il blocco si attiva o si libera, senza dover guardare lo schermo.
- Quando ti fermi, dopo qualche minuto puoi terminare il viaggio con un tocco.

### Filosofia: le tre assenze

- **Nessuna pubblicità** — la tua attenzione non è in vendita.
- **Nessuna sorveglianza** — non ti guardiamo, ti proteggiamo.
- **Nessun segreto** — il codice è aperto, perché la fiducia si prova, non si chiede.

### Perché open source

Φάρος chiede il permesso di vedere il traffico dati del telefono. Un'app così non deve chiedere fiducia sulla parola: deve darne la prova. Il codice pubblico è quella prova — chiunque può leggerlo e verificare che l'app fa solo ciò che promette.

La licenza **GPL-3.0** garantisce che il codice resti libero per sempre: chiunque può usarlo e modificarlo, ma ogni versione derivata deve restare a sua volta open source.

### Stato del progetto

Φάρος è in sviluppo attivo. Questo repository contiene il codice sorgente completo (Flutter/Dart per l'interfaccia, Kotlin nativo per il servizio di localizzazione e la VPN locale).

## 🇬🇧 English

### What is Φάρος

Φάρος is a free, open-source Android app that **automatically blocks mobile data traffic** while you drive, through a purely local VPN. No servers, no data collection, no advertising.

Like a lighthouse: it chases no one, makes no sound, demands nothing. It stands there, gives its light, and whoever sails uses it when needed.

### How it works

- Detects driving via GPS: speed above threshold for 15 seconds **and** real displacement of at least 100 metres (a double condition against false activations while stationary).
- Activates a local VPN that blocks data traffic for social media, messaging and video apps.
- **Navigation and music apps always stay free** — the "ears yes, eyes no" criterion: whatever can be used without taking your eyes off the road is allowed.
- A calm voice announcement tells you when the block activates or is released, without needing to look at the screen.
- Once stopped, after a few minutes you can end the trip with a tap.

### Philosophy: the three absences

- **No advertising** — your attention is not for sale.
- **No surveillance** — we do not watch you, we protect you.
- **No secrets** — the code is open, because trust is proven, not requested.

### Why open source

Φάρος asks permission to see the phone's data traffic. An app like this shouldn't ask for trust on faith: it should provide proof. The public code is that proof — anyone can read it and verify the app does only what it promises.

The **GPL-3.0** license ensures the code stays free forever: anyone can use and modify it, but any derived version must itself remain open source.

### Project status

Φάρος is under active development. This repository contains the full source code (Flutter/Dart for the interface, native Kotlin for the location service and local VPN).
