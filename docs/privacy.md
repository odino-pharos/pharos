## Informativa sulla Privacy — Φάρος (Pharos)

**Ultimo aggiornamento:** 8 luglio 2026

### Chi siamo

Φάρος (Pharos) è un'applicazione Android gratuita, open source e senza pubblicità, sviluppata da Gilberto Odino Pasquariello ("lo Sviluppatore"). Il codice sorgente completo è pubblicamente disponibile e verificabile su GitHub: [github.com/odino-pharos/pharos](https://github.com/odino-pharos/pharos).

Questa informativa spiega in modo chiaro e completo come Pharos tratta i dati durante il suo utilizzo.

### Il principio fondamentale

**Pharos non raccoglie, non trasmette e non condivide alcun dato personale con lo Sviluppatore o con terze parti.** Tutti i dati generati dall'app restano esclusivamente memorizzati sul dispositivo dell'utente, nella memoria locale dell'applicazione. Nessun server esterno riceve mai informazioni da Pharos.

### Dati trattati dall'app e dove restano

**Posizione GPS.** Pharos utilizza la posizione GPS del dispositivo (inclusa in background) per rilevare la velocità di movimento e determinare quando attivare o disattivare il blocco dei dati mobili. Le coordinate vengono elaborate esclusivamente in tempo reale sul dispositivo e non vengono mai salvate, registrate, trasmesse o condivise in alcuna forma.

**Sessioni di guida.** Al termine di ogni sessione, l'app salva localmente (storage privato sul dispositivo) durata del tragitto, chilometri percorsi, orario di inizio e fine. Questi dati restano solo sul dispositivo, non vengono mai sincronizzati o inviati altrove, e sono cancellabili in qualsiasi momento disinstallando l'app o cancellando i dati dell'app dalle impostazioni Android.

**Funzione VPN locale.** Pharos utilizza l'API VpnService di Android per bloccare selettivamente il traffico dati di alcune app durante la guida (eccetto navigazione e musica). Questa funzione opera interamente in locale: non instrada, non intercetta, non registra e non trasmette alcun traffico verso server dello Sviluppatore o terzi.

### Permessi richiesti

| Permesso | Finalità | Dati inviati a server esterni |
|---|---|---|
| Posizione (precisa/approssimativa/background) | Rilevare la velocità | Nessuno |
| Servizio in primo piano (localizzazione) | Monitoraggio GPS durante la guida | Nessuno |
| Notifiche | Mostrare lo stato di Pharos | Nessuno |
| Vibrazione | Feedback tattile | Nessuno |
| Avvio automatico al riavvio | Funzionamento continuo | Nessuno |
| Esclusione ottimizzazione batteria | Evitare interruzioni durante la guida | Nessuno |
| Accesso VPN locale | Blocco dati selettivo come sopra descritto | Nessuno |
| Internet | Riservato a future funzionalità (es. verifica aggiornamenti); attualmente inutilizzato | Nessuno attualmente |

Pharos **non richiede** l'accesso a contatti, fotocamera, microfono, SMS, file personali o account Google.

### Servizi di terze parti

Pharos **non integra alcun SDK di terze parti**: nessun analytics, nessun crash reporting, nessuna pubblicità, nessun tracciamento. L'elenco completo delle librerie è pubblico nel file `pubspec.yaml` del repository GitHub.

### Account utente

Pharos **non richiede alcuna registrazione, login o account**.

### Donazioni

Pharos offre facoltativamente la possibilità di sostenere il progetto tramite Ko-fi o PayPal. Queste transazioni avvengono interamente su tali piattaforme esterne, secondo le loro rispettive informative; Pharos non riceve né elabora dati di pagamento.

### Minori

Pharos non è rivolta specificamente a minori e non raccoglie consapevolmente dati di minori, coerentemente col fatto che non raccoglie dati di alcun utente.

### Diritti dell'utente (GDPR)

Poiché Pharos non raccoglie né conserva alcun dato personale su server esterni, l'utente ha già il pieno controllo dei propri dati, che risiedono unicamente sul proprio dispositivo. Disinstallare l'app o cancellare i suoi dati rimuove definitivamente ogni informazione generata da Pharos.

### Modifiche a questa informativa

Eventuali aggiornamenti futuri saranno pubblicati su questa stessa pagina, con la data aggiornata in cima al documento.

### Contatti

Per domande su questa informativa: [github.com/odino-pharos/pharos](https://github.com/odino-pharos/pharos).

## Privacy Policy — Φάρος (Pharos)

**Last updated:** July 8, 2026

### About Us

Φάρος (Pharos) is a free, open-source, ad-free Android application developed by Gilberto Odino Pasquariello ("the Developer"). The complete source code is publicly available on GitHub: [github.com/odino-pharos/pharos](https://github.com/odino-pharos/pharos).

### The Core Principle

**Pharos does not collect, transmit, or share any personal data with the Developer or with third parties.** All data generated by the app remains stored exclusively on the user's device. No external server ever receives information from Pharos.

### Data Handled by the App and Where It Stays

**GPS Location.** Pharos uses the device's GPS location (including in the background) to detect movement speed and determine when to activate or deactivate mobile data blocking. Coordinates are processed exclusively in real time on the device and are never saved, logged, transmitted, or shared.

**Driving Sessions.** At the end of each session, the app locally saves (private storage on the device) trip duration, kilometers traveled, and start/end time. This data stays on the device only, is never synced or sent elsewhere, and can be deleted anytime by uninstalling the app or clearing its data from Android settings.

**Local VPN Function.** Pharos uses Android's VpnService API to selectively block data traffic from certain apps while driving (except navigation and music apps). This operates entirely locally: it does not route, intercept, log, or transmit any traffic to Developer or third-party servers.

### Permissions Requested

| Permission | Purpose | Data sent to external servers |
|---|---|---|
| Location (fine/coarse/background) | Detect movement speed | None |
| Foreground service (location) | GPS monitoring while driving | None |
| Notifications | Show Pharos status | None |
| Vibration | Haptic feedback | None |
| Auto-start on boot | Continuous operation | None |
| Battery optimization exemption | Prevent interruptions while driving | None |
| Local VPN access | Selective data blocking as described above | None |
| Internet | Reserved for future features (e.g. update checks); currently unused | None currently |

Pharos **does not request** access to contacts, camera, microphone, SMS, personal files, or Google accounts.

### Third-Party Services

Pharos **does not integrate any third-party SDK**: no analytics, no crash reporting, no advertising, no tracking. The complete list of libraries is publicly available in the `pubspec.yaml` file of the GitHub repository.

### User Accounts

Pharos **does not require any registration, login, or account**.

### Donations

Pharos optionally offers the possibility to support the project via Ko-fi or PayPal. These transactions take place entirely on those external platforms; Pharos does not receive or process any payment data.

### Children

Pharos is not specifically directed at children and does not knowingly collect data from minors, consistent with the fact that it does not collect data from any user.

### User Rights (GDPR)

Since Pharos does not collect or store any personal data on external servers, the user already has full control over their own data, which resides solely on their device. Uninstalling the app or clearing its data permanently removes all information generated by Pharos.

Future updates will be published on this same page, with the last-updated date at the top of the document.
